# 三维空间分析 — 项目文件说明

> 基于 **Vite + Vue 3 + CesiumJS** 构建的 GIS 空间分析平台。

---

## 工程配置层

| 文件 | 用途 |
|------|------|
| `package.json` | 依赖声明（Cesium 1.142、Vue 3.5、Element Plus、Pinia、Vue Router）+ npm scripts（dev / build / preview） |
| `package-lock.json` | 依赖版本锁定，保证团队安装一致性 |
| `vite.config.js` | Vite 构建配置：Vue 插件、Cesium 插件、路径别名（`@/`→`src/`）、开发服务器端口 3000、3D Tiles 二进制文件的 MIME 类型设置 |
| `index.html` | 应用入口 HTML，挂载 `<div id="app">`，加载 Google Fonts（Rajdhani + Noto Sans SC + JetBrains Mono），引入 `src/main.js` |
| `.env` | 环境变量：高德 API Key（`VITE_AMAP_KEY`、`VITE_AMAP_JSCode`），Vite 构建时注入 |
| `skills-lock.json` | Claude Code 技能锁定文件（frontend-design 技能的版本与哈希记录） |
| `README.md` | 项目说明文档：功能模块、技术架构、项目结构、API 配置、文档索引 |

## `src/` — 应用源码

### 入口

| 文件 | 用途 |
|------|------|
| `src/main.js` | 创建 Vue 应用实例 → 依次挂载 Pinia（状态管理）、Vue Router（路由）、Element Plus（中文语言包 + 暗色主题）→ 加载全局样式 `main.css` → 挂载到 `#app` |

### 根组件

| 文件 | 用途 |
|------|------|
| `src/App.vue` | **全局 Shell 布局**。核心职责：① Cesium 三维场景容器 `#cesiumContainer`（挂载时调用 `useCesium` 初始化 Viewer）；② 侧边导航栏（品牌区 SVG Logo / 6 项路由菜单 / 系统时钟 + 运行状态）；③ `<router-view>` 面板过渡动画（`panel-trans` fade + slide）。侧边栏支持折叠，影像/地形类型通过 Pinia Store 跨路由持久化 |

### 路由

| 文件 | 用途 |
|------|------|
| `src/router/index.js` | 6 条路由定义（Hash 模式 `createWebHashHistory`），全部使用**懒加载**（动态 `import()`），每条路由携带 `meta` 元信息（中文标题 + 图标 emoji） |

### 状态管理

| 文件 | 用途 |
|------|------|
| `src/stores/viewer.js` | **Pinia Store** — 全局单例状态：① `viewer` — 全局唯一的 Cesium Viewer 实例引用；② `isReady / isLoading / error` — 初始化状态追踪；③ `activeTool / statusMessage / statusType` — 当前激活工具与状态栏信息；④ `imageryType / terrainType` — 影像/地形选择持久化（跨路由保持）；⑤ `_trackedDataSources` — 内部 DataSource 追踪 Map（供 `useCesium` 销毁时遍历清理） |

### 组合式函数（Composables）

| 文件 | 用途 |
|------|------|
| `src/composables/useCesium.js` | **Viewer 生命周期管理**。`useCesium()` — 挂载时注入 Cesium Ion Token、创建 Viewer 实例（关闭时间轴/动画/底图选择器等非必要控件）、场景参数配置（光照/地形深度检测/雾效/屏幕空间误差）、隐藏版权信息、错误捕获；卸载时遍历清理所有 DataSource、销毁 Viewer、重置 Store。`useViewer()` — 从 Pinia Store 获取响应式 Viewer 引用（使用 `storeToRefs` 保持响应性） |
| `src/composables/useImagery.js` | **底图/地形切换**。`useImagery()` — 三种影像底图切换：Cesium 全球影像（`createWorldImageryAsync`）、高德卫星影像（URL 模板 `https://webst01.is.autonavi.com/...`）、腾讯卫星影像（URL 模板含自定义 `customTags` 用于 subdomain 切分）；`useTerrain()` — 三种地形模式：Cesium 全球地形（`createWorldTerrainAsync`）、ArcGIS 高程服务、扁平地球（`EllipsoidTerrainProvider`）。所有切换操作通过 Pinia Store 读写当前类型 |
| `src/composables/useCoordinate.js` | **坐标转换工具**。`useCoordinate()` — 屏幕坐标拾取（`camera.getPickRay` → `globe.pick`）、十进制度→度分秒格式化、经纬度↔Cartesian3 互转、bbox→Rectangle；`CoordinateConverter` 静态类 — **GCJ-02 ↔ WGS-84 双向转换**（中国测绘局偏转算法，含中国境外检测，自动跳过无偏转区域） |

### 服务层（Services）

| 文件 | 用途 |
|------|------|
| `src/services/trafficService.js` | **高德交通态势 API**。`TrafficService` 类：① 矩形区域查询（`fetchTrafficByBBox`）— WGS-84 坐标自动转 GCJ-02 请求，超 10km 对角线自动沿长边方向二拆分，最多 4 路并发 + 去重（按 road.id）；② 道路名查询（`fetchTrafficByRoad`）— 需同时传城市名，无结果时自动全国重试；③ GCJ-02 底图适配 — `keepGcj02` 选项在检测到高德/腾讯影像时跳过坐标转回，保证路况线与底图对齐；④ 缓存 — 5 分钟 TTL，按 bbox 精度 4 位小数做 key；⑤ 静态工具 — `diagonalMeters()` 供 UI 层实时计算对角线距离 |
| `src/services/routeService.js` | **高德路径规划 API**。`RouteService` 类：① WGS-84 坐标自动转 GCJ-02 请求 + 路径点坐标转回 WGS-84；② 标准化输出 — 距离/时间/过路费/红绿灯数/分步 polyline；③ 可取消 — 内部 `AbortController`，切换路由时自动取消；④ 指数退避重试 — 最多 2 次重试（1s / 2s），4xx 不重试；⑤ 关键错误码中文映射 |

### 工具层（Utils）

| 文件 | 用途 |
|------|------|
| `src/utils/cesiumHelpers.js` | **Cesium Entity 工厂函数 + 通用工具**。`cesiumHelpers` 对象：① `createPointEntity()` — 点标记（含 label，支持贴地/空间）；② `createLineEntity()` — 折线（支持实线/发光/虚线三种材质）；③ `createPolygonEntity()` — 多边形（半透明填充+描边）；④ `createRectangleEntity()` — bbox 矩形；⑤ `removeEntities()` — 安全移除单个或批量 Entity；⑥ `flyTo()` — 计算包围球并飞行；⑦ `createDataSource()` — 创建并注册 CustomDataSource；⑧ `destroyHandler()` — 安全销毁 ScreenSpaceEventHandler；⑨ 颜色常量 — CYAN/GREEN/AMBER/RED/BLUE 预定义 + 交通态势四级色板 |

### 全局样式

| 文件 | 用途 |
|------|------|
| `src/styles/main.css` | **航天气质暗色设计系统**。① CSS 自定义属性 — 四层深色变量（`--void-deep/mid/surface/raised`）、HUD 霓虹色（cyan/amber/green/red）、字体三件套（Rajdhani 标题 + Noto Sans SC 正文 + JetBrains Mono 数据）；② Element Plus 暗色主题完整覆写（主色/背景/边框/文字全部替换为 HUD 色系）；③ 全局 Reset + 氛围背景（CSS 网格线 + SVG 噪点纹理）；④ 自定义滚动条；⑤ 玻璃面板组件 `.glass-panel`（毛玻璃+霓虹边框+折叠动画）；⑥ HUD 标签 `.hud-label`（带渐变分割线）；⑦ 坐标叠加层 `.coord-overlay`（右下角悬浮）；⑧ 按钮宽度归一化修复（解决 `el-button` 的 `inline-flex` 导致 `width:100%` 不生效的问题）；⑨ 侧边栏菜单微调；⑩ 脉冲发光 + 淡入上浮动画关键帧 |

### 视图（Views）

| 文件 | 用途 |
|------|------|
| `src/views/Home.vue` | **基础三维场景**。功能：① 底图切换（Cesium/高德/腾讯三种卫星影像）；② 地形切换（扁平/Cesium/ArcGIS 三种模式）；③ 视角控制（俯视/45°斜视/平视三种预设 + 全局复位）；④ 视场角滑块（30°~90°）；⑤ glTF 模型加载（Cesium 飞机，屏幕中心定位）；⑥ 3D Tiles 加载（从 tileset.json 解析包围盒自动飞行定位）；⑦ GeoJSON 加载（武汉市行政区划，TERRAIN 分类叠加）；⑧ 分色渲染（基于 adcode 哈希配色 + 白色描边 + 区域注记 + 色卡图例）；⑨ 高度可视化（批量地形采样 → 挤出柱状体 → 高度标注 → 统计卡片，首次采样后缓存复用）；⑩ 鼠标点击坐标拾取（笛卡尔/经纬度/度分秒实时显示） |
| `src/views/TrafficVisualization.vue` | **交通态势可视化**。三种查询模式：① 视野查询 — 监听相机移动实时计算视野 bbox 及对角线距离，超 10km 自动拆分提示；② 手绘矩形 — 两个角点绘制矩形预览，自动换算对角线；③ 道路名联想搜索 — 三阶段策略（高德 Input Tips → Place Search Text → 后缀并行扩展如"中关村"→"中关村大街"），每条结果显示道路名+所属城市+置信度标签（主干/道路），键盘上下选择+Enter 确认；渲染 — 畅通绿/缓行黄/拥堵红/严重拥堵深红四级配色，线宽随速度动态变化；GCJ-02 底图自动检测适配 |
| `src/views/ViewshedAnalysis.vue` | **可视域分析**。功能：① 观察点设置（地图点选 / 手动输入经纬度+离地高度）；② 360° 方向 × 每向 10 步采样点 → 批量 `sampleTerrainMostDetailed` → 逐射线 LOS 判定（线性 LOS 高度 vs 地形高度 + 0.5m 容差）；③ 结果统计（可见面积 km² / 可见比例 %）；④ 导出 JSON（含完整 360 射线网格数据）；⑤ 导出 GeoJSON（Polygon，可在 QGIS/Leaflet 加载）；⑥ JSON 回显（导入之前导出的分析结果重新可视化）；⑦ 截图保存（canvas.toDataURL） |
| `src/views/BufferAnalysis.vue` | **缓冲区分析**。两种模式：① 点缓冲 — 点击选点，72 段正圆近似（按纬度修正经度方向米-度换算），贴地多边形 `classificationType: TERRAIN`；② 线缓冲 — 多点点击绘制（Enter 完成 / Esc 取消），标准 GIS 算法（线段左右平移 + 拐点斜接 + 起点/终点半圆圆帽），最终多边形由 `起点帽 → 左侧 → 终点帽 → 右侧逆序` 组装闭合环 |
| `src/views/RoutePlanning.vue` | **路径规划分析**。功能：① 起终点设置（地图点选 / 手动输入经纬度，分别高亮青色/琥珀色标记）；② 高德驾车路径查询（`RouteService.fetchDrivingRoute`）；③ 发光路径渲染（`PolylineGlowMaterialProperty` + 距离标签）；④ 结果统计卡片（公里数/预估时间/红绿灯数） |
| `src/views/SightlineAnalysis.vue` | **视线分析**。功能：① 观察点→目标点有向分析（依次点选 + 重新选择按钮）；② **GIS 标准最大斜率 LOS 算法**（与 ArcGIS/QGIS/GRASS 一致）— 从观察者逐点计算 `slope = (terrainH - observerH) / groundDistance`，维护 `maxSlope`，大于者可见并更新；③ 三段可视化 — 底层参考线（白色半透明）+ 可见段（绿色发光）+ 遮挡段（红色虚线）+ 首遮挡点（红色三角标记）+ 稀疏遮挡点（黄色小点，最多 15 个）；④ 采样密度/观察点离地高度/目标点离地高度均可调 |

## 数据流总览

```
用户操作（面板按钮 / 地图点击）
        │
        ▼
  Vue 视图组件（6 个 .vue 文件）
        │
        ├──→ useCesium / useViewer      （获取 Viewer 实例）
        ├──→ useImagery / useTerrain     （切换底图地形）
        ├──→ useCoordinate              （坐标拾取与转换）
        ├──→ RouteService               （高德路径规划 API）
        ├──→ TrafficService             （高德交通态势 API）
        └──→ cesiumHelpers              （创建 / 清理 Entity）
                │
                ▼
          Cesium Viewer                （三维渲染引擎）
                │
                ▼
           Pinia Store                  （全局状态共享）
```

    ├── RoutePlanning.vue         路由 /route-planning
    └── SightlineAnalysis.vue     路由 /sightline
